computeFormation (args)
Computes a formation for a unit.

Parameters:

  • args The `name`, `bounds`, `count`, and `position` to compute.

Returns:

    A list of positions to use per mini-index.

Usage:

    computeFormation({
  -- Optional. Defaults to `2-line`.
  name = '2-line',
  bounds = {1, 1, 1},
  count = 4,
  position = {0, 0, 0},
})
